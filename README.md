# 面接無双（デモ版）

就活の面接練習Webアプリ。一次〜最終面接を、日本人面接官との対話形式で練習できます。

- 業界6種×面接段階3種＋主要11社の実出題データ（就活体験談ベース・2026年7月編集）
- 回答内容に反応する相槌・深堀り連鎖
- 5観点スコアと添削レポート、評価理由をコーチに質問できるチャット
- 壁打ち（書き直し再添削）とLIKE!復習ノート（記録はブラウザのlocalStorageに保存）

## 技術
単一HTMLファイルの静的サイト（ビルド不要・外部通信なし）。

## クレジット
- 写真素材: [ぱくたそ](https://www.pakutaso.com/)（利用規約に基づき使用）
- 質問データ: 就活情報サイトで報告された面接体験談をもとに編集（詳細はアプリ内フッター参照）
