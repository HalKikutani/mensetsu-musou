## 横断的に使える評価知識（添削エンジン強化の核）

- **NG表現辞書**（複数ソースから集約）: 「頑張りました」「一生懸命」「チームで◯◯しました（主語なし）」「グローバルに活躍したい」「安定しているから」「人の役に立ちたい」「ITが好き」「成長したい」「大変でした」→ いずれも具体化を促す改善指摘文とセットで収録済み
- **黄金比**: ガクチカ・自己PRは200〜400字（ES基準。面接口頭では現行LEN_RULESと整合を取る）
- **STAR完全性**: S=期間・場所・規模の数字、T=課題と原因、A=「私は」主語＋手法名、R=ビフォーアフター数値＋外部評価
- **弱みの正解パターン**: 弱み宣言→具体例→改善行動→現状（改善行動なしの弱み回答は減点対象にできる）
- **難易度体系**: ★☆☆定番/★★☆要準備/★★★深掘り・圧迫対策必須 → 面接段階（一次/二次/最終）との対応付けに使える
